# disney-hotstar-clone

Hotstar-Disney+ clone using ReactJS

## Installation

need to install git and npm in system

```
git clone https://github.com/DavidLivingstoneHini/disney-hotstar-clone.git

cd disney-hotstar-clone

npm i

npm start

```
